Datos que Recopilamos

Recopilamos información que nos proporcionas directamente:

  • Datos de identificación: nombre, apellido, email, teléfono, país
  • Datos de uso: actividad en la plataforma, operaciones realizadas
  • Datos técnicos: dirección IP, tipo de dispositivo, navegador
  • Documentos KYC: cuando los proporcionas voluntariamente

Cómo Usamos tus Datos

Utilizamos tu información para:

  • Proveer y mejorar continuamente el servicio
  • Verificar tu identidad y prevenir fraudes
  • Enviarte comunicaciones importantes sobre tu cuenta
  • Cumplir con obligaciones legales y regulatorias
  • Analizar el uso de la plataforma para mejoras

Seguridad de Datos

Implementamos medidas de seguridad de nivel bancario:

Encriptación AES-256
Autenticación 2FA
Acceso restringido
Monitoreo 24/7

Tus Derechos

Como usuario tienes derecho a:

  • Acceso: Solicitar una copia de tus datos personales
  • Rectificación: Corregir datos inexactos o incompletos
  • Eliminación: Solicitar la eliminación de tus datos
  • Portabilidad: Recibir tus datos en formato estructurado
  • Oposición: Oponerte al procesamiento de tus datos
Para ejercer estos derechos, contáctanos en: privacidad@novacapitalgroup.com

Compartir Datos con Terceros

No vendemos ni alquilamos tus datos personales a terceros. Solo compartimos información cuando es necesario para proveer el servicio, cumplir con la ley o proteger nuestros derechos legales.
